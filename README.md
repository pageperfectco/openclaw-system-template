# OpenClaw System Template

A ready-to-run multi-agent OpenClaw setup. Clone this, follow `INSTALL.md`, and your agents are live.

## What's Included

| Agent | Role | Workspace |
|-------|------|-----------|
| **Felix** | CEO / Orchestrator | `workspace/` |
| **Iris** | Developer | `workspace-dev/` |
| **Kat** | Master Copywriter | `workspace-kat/` |
| **Alex** | Chief Growth Officer | `workspace-alex/` |
| **Nova** | Customer Success | `workspace-nova/` |
| **Pierce** | QC & Security | `workspace-pierce/` |

## What This Gives You

- Multi-agent dispatch protocol (Felix queues work, agents execute, status tracked)
- Copy always routes through Kat — dev agents never write customer-facing copy
- Growth & CRO owned by Alex — Felix doesn't touch marketing strategy
- QC/security audits owned by Pierce
- Customer success owned by Nova
- Task queue + agent status scripts so nothing gets lost across restarts

## Quick Start

Read `INSTALL.md` — it's written to be followed by both humans and AI agents.
